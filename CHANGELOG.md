# Changelog

[English](./CHANGELOG.md) | [中文](./CHANGELOG.zh-CN.md)

All notable changes for this repository are documented in this file.

## 2026-03-14 (Beta)

### Added
- Conversational analytics workflow with diagnostics and failure fallback suggestions.
- Core dashboards for sales, operations, and agile analytics.
- Chart catalog and chart validation gate integrated into CI workflow.
- AI governance pages for model status, alerts, thresholds, and switch history.
- Release readiness scripts: deployment drill, security checks, support checks.

### Known Limitations
- Current release is Beta and not a final production capacity commitment.
- Some complex cross-domain semantic prompts may still require clarifying questions.
- Visual baseline checks are sensitive to layout changes and need synchronized baseline updates.
- Performance baselines are validated in current environment only.
