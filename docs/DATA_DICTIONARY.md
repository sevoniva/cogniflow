# Data Dictionary (Summary)

[English](./DATA_DICTIONARY.md) | [中文](./DATA_DICTIONARY.zh-CN.md)

This is the external-facing summary of data entities used by Chat BI.

## Main Domain Areas

- Identity and permissions
- Data source and metadata registry
- Metrics and dashboards
- Query history and diagnostics
- AI model config and observability

## Naming and Conventions

- Relational storage is MySQL-compatible.
- Audit/time fields are expected for core entities.
- API responses use a unified envelope (`code/message/data/timestamp`).
