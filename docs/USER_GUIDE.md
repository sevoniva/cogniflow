# User Guide

[English](./USER_GUIDE.md) | [中文](./USER_GUIDE.zh-CN.md)

## What Chat BI Delivers

- Conversational analytics from natural language to governed query
- Dashboard and chart rendering with validation gate support
- AI model observability, alerting, and switch history

## Typical User Flow

1. Open Conversation page and ask a business question.
2. Review generated diagnostics and SQL intent.
3. Render chart/table result and refine the query.
4. Track model and alert status in AI observability pages.
