## 1. 产品简介

### 1.1 什么是 ChatBI?

ChatBI 是一款企业级智能 BI 分析平台，通过自然语言对话的方式，让用户能够轻松进行数据查询和分析。

### 1.2 核心特性

- **自然语言查询**: 用日常语言提问，自动转换为 SQL 查询
- **可视化仪表板**: 拖拽式编辑器，轻松创建专业仪表板
- **智能分析**: AI 驱动的数据洞察和异常检测
- **企业级安全**: 行级权限控制、数据脱敏、LDAP 集成
- **多数据源支持**: MySQL、Oracle、PostgreSQL、ClickHouse 等

### 1.3 适用场景

- 业务人员自助数据分析
- 管理层决策支持
- 运营数据监控
- 财务报表生成
- 销售数据分析

## 2. 快速开始

### 2.1 注册登录

1. 访问系统首页
2. 点击"注册"创建账号
3. 使用用户名密码登录

### 2.2 首次使用

#### 步骤 1: 配置数据源

1. 进入管理后台
2. 选择"数据源管理"
3. 点击"新建数据源"
4. 填写数据库连接信息
5. 测试连接并保存

#### 步骤 2: 定义指标

1. 进入"指标管理"
2. 点击"新建指标"
3. 选择数据表和字段
4. 设置聚合方式
5. 保存指标

#### 步骤 3: 开始查询

1. 返回首页
2. 在查询框输入问题，如"查看本月销售额"
3. 系统自动生成查询并展示结果

## 3. 核心功能使用

### 3.1 智能查询

#### 自然语言查询

```
示例 query:
- "查看上个月的销售额"
- "哪个地区的销量最好"
- "显示近 7 天的用户活跃度趋势"
- "对比今年和去年的利润"
```

#### 查询结果操作

- **导出**: 支持 Excel、CSV、PDF 格式
- **分享**: 生成分享链接或二维码
- **收藏**: 保存常用查询
- **下钻**: 点击数据查看详细

### 3.2 可视化查询构建器

#### 使用步骤

1. 进入"可视化查询"页面
2. 选择数据源和数据表
3. 勾选需要查询的字段
4. 添加过滤条件
5. 设置分组和聚合
6. 点击"执行查询"

#### 支持的查询操作

| 操作 | 说明 |
|------|------|
| 字段选择 | 选择需要查询的列 |
| 过滤条件 | =, !=, >, <, LIKE, IN 等 |
| 分组聚合 | GROUP BY + SUM/COUNT/AVG/MAX/MIN |
| 排序 | ASC/DESC |
| 限制条数 | LIMIT |

### 3.3 仪表板管理

#### 创建仪表板

1. 进入"仪表板管理"
2. 点击"新建仪表板"
3. 输入名称和描述
4. 点击"编辑"进入编辑器

#### 编辑仪表板

1. **添加组件**: 从左侧拖动图表类型到画布
2. **配置数据**: 选择数据表、字段、聚合方式
3. **调整布局**: 拖拽调整位置和大小
4. **设置样式**: 配置颜色、标题、图例等

#### 支持的图表类型

| 图表 | 适用场景 |
|------|----------|
| 柱状图 | 对比数据 |
| 折线图 | 趋势分析 |
| 饼图 | 占比分析 |
| 散点图 | 相关性分析 |
| 热力图 | 密度分布 |
| 桑基图 | 流量分析 |
| 漏斗图 | 转化分析 |
| 地图 | 地理分布 |

### 3.4 数据权限管理

#### 配置行级权限

1. 进入"数据权限"页面
2. 点击"新建权限规则"
3. 选择表名和字段
4. 设置操作符和值
5. 选择值类型（常量/字段/用户属性）
6. 设置优先级和状态

#### 权限规则示例

| 规则 | 表 | 字段 | 操作 | 值 | 说明 |
|------|-----|------|------|-----|------|
| 部门隔离 | sales | dept_id | = | 用户部门 | 用户只能看本部门数据 |
| 区域限制 | orders | region | IN | 华东，华南 | 限制查看区域 |
| 数据脱敏 | users | phone | 部分隐藏 | 前 3 后 4 | 手机号脱敏 |

### 3.5 数据脱敏

#### 脱敏类型

| 类型 | 说明 | 示例 |
|------|------|------|
| HIDE | 完全隐藏 | *** |
| PARTIAL | 部分隐藏 | 138****1234 |
| HASH | SHA-256 哈希 | 5f4dcc3b... |
| ENCRYPT | AES 加密 | (密文) |

#### 配置脱敏规则

1. 进入"数据脱敏"页面
2. 点击"新建脱敏规则"
3. 选择表和字段
4. 选择脱敏类型
5. 设置脱敏模式（如"前 3 后 4"）
6. 保存规则

### 3.6 告警规则

#### 创建告警

1. 进入"告警规则"页面
2. 点击"新建告警规则"
3. 填写规则名称
4. 选择告警类型
5. 设置阈值
6. 选择通知方式
7. 填写接收人

#### 告警类型

| 类型 | 说明 | 配置项 |
|------|------|--------|
| THRESHOLD | 阈值告警 | 大于/小于/等于 + 阈值 |
| FLUCTUATION | 波动告警 | 波动率% |
| ANOMALY | 异常检测 | AI 自动检测 |

### 3.7 查询历史与收藏

#### 查看历史

1. 进入"查询历史"页面
2. 查看最近的查询记录
3. 支持关键词搜索
4. 点击可重新执行查询

#### 管理收藏

1. 进入"我的收藏"页面
2. 查看收藏的查询
3. 支持重命名
4. 点击快速执行

### 3.8 下钻/上卷分析

#### 使用下钻

1. 在查询结果页面
2. 点击数据项
3. 选择"下钻"
4. 选择下钻维度（年→季度→月→日）

#### 使用上卷

1. 在下钻后的页面
2. 点击面包屑导航
3. 选择上级维度
4. 查看汇总数据

### 3.9 嵌入式分析

#### 创建嵌入

1. 进入仪表板页面
2. 点击"分享"按钮
3. 选择"嵌入"
4. 设置有效期
5. 生成嵌入代码

#### 嵌入到第三方

```html
<!-- 复制以下代码到您的网站 -->
<iframe
  src="https://chatbi.example.com/embed/abc123"
  width="100%"
  height="600"
  frameborder="0">
</iframe>
```

## 5. 常见问题

### 5.1 查询相关

**Q: 为什么查询结果为空？**
A: 检查过滤条件是否正确，数据表中是否有数据

**Q: 查询速度慢怎么办？**
A: 添加索引、优化 SQL、使用缓存

### 5.2 权限相关

**Q: 为什么看不到某些数据？**
A: 检查行级权限配置

**Q: 如何给用户授权？**
A: 在权限配置中分配角色

### 5.3 仪表板相关

**Q: 如何修改仪表板布局？**
A: 进入编辑模式，拖拽组件调整

**Q: 仪表板如何分享？**
A: 点击分享按钮，生成链接或嵌入代码
